# **🐤 Difference of Parameter, Argument**

## **매개변수(Parameter), 전달인자(Argument)란?**
**Parameter**
> ```
> int sum(int a, int b){
>   return a+b;
> }
> ```
> (int a, int b)는 실제 값이 존재하지 않고 형태를 나타내줄 뿐.  
> 이것을 "Parameter", 매개변수라고 한다.

**Argument**
> ```
> sum(10, 20);
> ```
> 위의 sum 함수를 호출 했을 때, ```(10,20)``` 의 변수를 전달인자 라고 한다.  
> 전달인자에는 값이 존재한다.
